I started my journey from scratch - 0.00$. I used some faucets and played some rewarding games (there are some applications that pay you to play mobile/pc games) and gathered some basic 'resources' (in the form of crypto currency), which I invested in a few places, which seemed safe enough. The ROI is pretty low if you start with 0$, but it is a option, if you have free time and want to put in some effort. You need to collect at least 5$ if you are to invest in a small platform with the idea of generating some passive income (I would suggest atleast 15$). The more $ you invest - the better, but it's better to invest little in long-term platform with lesser ROI, than a high-return platform, which proves to be a scam. Usually, when I see a platform with ROI > 3% per day, I get suspicious.
